Meter Monthly Cost

Summary

This page provides data about how much each meter costs during every month at TCNJ. This data could be especially useful when considering what buildings each meter represents. For example, on average meter EL1 costs significantly more per month than any other electric meter. This would indicate that the building or buildings it represents may be able to cut down on costs.
Another interesting query is using the "Get all monthly averages for a certain meter. Choose a meter: " query to select Estimated Co-Gen Electric. This shows a breakdown of how much energy is produced through this method each month.
